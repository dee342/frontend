Here's what Frontend looks like… last updated February 17th, 2017

Thanks for checking out Frontend by Ben Marshall! Here you can take a look at the default styles and functionality Frontend comes with. Everything you see below is completely configurable by cloning the src/scss/framework/global/_variables.scss file and placing it in the src/scss/application/global directory. IMPORTANT: After you've cloned the file, be sure to delete the !default parameter and the preceeding spacing (leaving the ;) for all variables.

TIP: Keep this file as a style guide to refer to as you develop and test.
